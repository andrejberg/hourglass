## Install
### From source
`
git clone git@github.com:andrejberg/hourglass.git
cd hourglass
pip install -r requirements.py
python main.py
`
### Windows precompiled
- Download zip
- Extract
- Run main.exe

### Compile

python setup.py build

(cp -r platforms build/)
